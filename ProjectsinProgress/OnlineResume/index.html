Stock Recommendation Script (Python)

Problem

Investing can be intimidating, especially when theres hundreds of stocks to analyze. This can often deter new and old investors from seeking out financial freedom.

Solution

Script uses Python's Pandas and Robin Stock's API to analyze annual real-time and historic stock prices in order to identify trends and present opportunities for both long-term and short-term trading strategies. This tool is intended to minimize search time for investors and does not eliminate "due diligence", a fundamental skill learned through investing.
